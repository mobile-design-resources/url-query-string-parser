URL Query String Parser

Type a query string in the text box below, the click on "Show test page" to open the test page with the specified query string.

A query string always starts with a question mark ("?"), which is already provided for you. The question mark follows several key-value entries in the form of "key=value", each entry separated by the "&" sign.
Example: "test.html?a=123&b=456"
